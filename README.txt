****** Test on the toy dataset ******

1.	Modify the script prepare_toy_dataset.py to generate the toy data.
	New data files will be created in data/toy_data/ directory.

2. 	Modify the configuration parameters in the config.py file.

3.	Run the main file.
	python main.py

4.	Modify the configuration file (config.py) for the inference.

5.	To run the inference scripts.
	python importance_ltr_compare.py

Latent Dirichlet allocation (LDA) using collapsed Gibbs sampling was implemented in reference to https://github.com/lda-project/lda.
